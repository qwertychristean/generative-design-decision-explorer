## Prompt 1: Dashboard Visual Direction

Create a dark-mode motorsport engineering dashboard for comparing generative design outputs. The interface should show a suspension bracket design review, weight versus safety factor, material choices, manufacturing method, cost level, and risk notes. Use black, charcoal, white, and papaya orange accents. The layout should feel clean, technical, fast to scan, and suitable for a racing team stakeholder review.

## Prompt 2: Concept Visual for Generative Design Part

Create a conceptual image of a lightweight motorsport suspension mounting bracket inspired by generative design. The form should look organic, skeletal, and optimized for stress paths. Use titanium or aluminum material texture. Place it on a dark engineering presentation background with subtle orange accents. This is a concept visual only, not a validated engineering model.
