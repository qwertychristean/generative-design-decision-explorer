# Generative Design Decision Explorer

A self-directed Information Science and generative AI project created for a McLaren Racing NEXT 2026 application proof pack.

## Project Purpose

Generative design creates many possible engineering solutions based on defined constraints such as preserve geometry, obstacle geometry, loads, materials, and manufacturing methods.

This project focuses on the information layer that comes after those options are generated.

The goal is to show how AI-generated engineering outputs might be organized, compared, explained, and turned into clearer decisions.

## Project Scenario

A fictional motorsport team is evaluating design options for a lightweight front suspension mounting bracket.

The team needs to compare possible designs based on:

- Weight
- Factor of safety
- Material
- Manufacturing method
- Estimated cost
- Load case
- Risk notes

The dashboard helps a technical stakeholder compare options and decide which design deserves deeper review.

## Important Note

This is a concept project.

The design data is mock data created for portfolio demonstration. The parts shown are not validated engineering outputs.

The purpose is to demonstrate information architecture, AI workflow understanding, dashboard design, and technical communication.

Build for production:

