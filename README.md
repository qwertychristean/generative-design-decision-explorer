# Generative Design Decision Explorer

Live Project: https://generative-design-decision-explorer.vercel.app

A self-directed Information Science and generative AI project created as part of my McLaren Racing NEXT 2026 application proof pack.

## Project Purpose

Generative design uses AI-supported software to create many possible engineering design options based on constraints such as geometry, loads, materials, manufacturing methods, weight, safety, and cost.

This project focuses on the information layer after those options are created.

The goal is to show how AI-generated engineering options might be organized, compared, and communicated through a decision-support dashboard.

## Project Scenario

A fictional motorsport team is reviewing design options for a front suspension mounting bracket.

The team needs to compare each option based on:

* Weight
* Safety factor
* Material
* Manufacturing method
* Estimated cost
* Lead time
* Risk notes
* Decision score

The dashboard helps a technical stakeholder quickly compare options and identify which design deserves deeper engineering review.

## Important Note

This is a concept project using mock data.

The design options shown are not validated engineering outputs. The project is meant to demonstrate information architecture, generative design workflow understanding, dashboard design, and technical communication.
